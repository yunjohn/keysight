# 本地离线测试建议

适用场景：当前手边没有示波器，希望先验证 UI、波形分析和交互逻辑。

## 启动

```powershell
.venv\Scripts\python main.py
```

如果只做离线测试，不需要连接设备。

## 建议优先验证的功能

### 1. 单通道离线波形分析

1. 点击 `加载 CSV`
2. 选择：
   `captures\waveforms\CH1_NORMal_20260323_141001.csv`
3. 检查：
   - 波形是否正常显示
   - 当前视图关键指标是否有值
   - `游标 / 全局统计 / 当前视图统计` 是否正常刷新

### 2. 波形交互

建议逐项测试：

- 滚轮缩放
- `Shift + 滚轮` 仅缩放时间轴
- `Ctrl + 滚轮` 仅缩放电压轴
- 左键框选局部放大
- 右键双击重置
- `横向重置 / 纵向重置 / 重置视图`

### 3. 游标交互

建议逐项测试：

- `设置游标 A / B`
- 拖动竖线，仅改时间
- 拖动横线，仅改电压
- 拖动交点，同时改时间和电压
- 悬停时鼠标形状、图上标签、状态提示是否一致
- `智能锁定 / 锁定最近脉冲 / 锁定最近周期`

### 4. 独立窗口

1. 点击 `独立显示`
2. 检查主界面与独立窗口是否同步
3. 在独立窗口重复测试缩放、游标和锁定功能

### 5. 导出能力

建议测试：

- `导出图像`
- `导出 CSV`

输出目录：

- `captures\waveform_images\`
- `captures\waveforms\`
